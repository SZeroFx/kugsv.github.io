Project Overview

LME (lme.lu) is an ambitious school entrepreneurship project that we're transforming into a real business. What started as an academic initiative is now evolving into a full-fledged startup, combining our learning experience with real-world business development.

LME provides a Secure Multi-Tenant Hosting Platform built on Hetzner infrastructure, designed to offer small organizations and individual clients reliable web hosting without the complexity and cost of large cloud providers.

Visit LME Website

Technology Stack

Infrastructure & Virtualization

  • Hetzner Dedicated Server - Reliable German infrastructure provider
  • Proxmox - Enterprise-grade virtualization platform for VM management
  • Docker + Docker Compose - Container orchestration for service isolation

Web Services & Security

  • Nginx - High-performance web server and reverse proxy
  • Cloudflare - DNS management, DDoS protection, and security features
  • SSH & Firewall - Secure access and network protection

Monitoring & Management

  • Raspberry Pi - Independent external monitoring system
  • Hetzner API - Resource monitoring and server management
  • Custom Scripts - Automated availability checks and alerts

Technical Architecture

  • VM-Level Separation - Each client website runs in a dedicated Proxmox virtual machine
  • Container Isolation - Docker Compose separates services within each VM
  • Network Segmentation - Firewall rules prevent cross-tenant access
  • Security Measures - SSH key-based authentication, automated updates, regular backups

Business Development

LME represents a unique combination of academic learning and real-world entrepreneurship:

  • Project Management Course - Developed for the Talent Exhibition as part of our school curriculum
  • House of Entrepreneurship Partnership - Guidance from Luxembourg's entrepreneurship ecosystem
  • Team of Three - Collaborative approach combining technical expertise and business development
  • Market Focus - Targeting small organizations who need reliable hosting without complexity
