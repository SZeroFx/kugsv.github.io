LuxRP

Private

Project Overview

LuxRP is a Luxembourg-based FiveM roleplay server that provides an immersive Grand Theft Auto V multiplayer experience. FiveM is a modification framework for GTA V, enabling players to engage in custom multiplayer experiences on dedicated servers with unique gameplay mechanics and community-driven roleplay scenarios.

Visit LuxRP Website Watch Reportage

Technology Stack

  • FiveM - Multiplayer modification framework for GTA V built on CitizenFX
  • NUI (New UI) - HTML-based user interface system using Chromium Embedded Framework
  • HTML5, CSS3, JavaScript - Frontend development for in-game interfaces
  • Lua - Server scripting and game logic (handled by other team members)

My Role

I served as Developer, Supporter, and Administrator:

Developer

  • Custom UI Development - Designed in-game interfaces using HTML, CSS, JavaScript within the FiveM NUI framework
  • Responsive Design - Created interfaces adapting to different screen resolutions
  • Performance Optimization - Ensured UIs were lightweight and performant

Supporter & Administrator

  • Conflict Resolution - Managed support situations with diplomacy and fairness
  • Player Assistance - Helped with technical issues and rule clarifications
  • Server Management - Player management, policy enforcement, team coordination

Featured Project: Funex

Funex is a dual-platform communication system I developed that bridges web and in-game environments. It operates both as a standalone website and as an integrated in-game phone application.

Technical Implementation

  • Tech Stack - HTML, CSS, JavaScript, PHP, MySQL, AJAX
  • Dual Environment - Works as web app and in-game mobile phone interface
  • Character Sync - Automatic synchronization with game database
  • Multi-Character Support - Each player can have multiple roleplay characters
