Project Overview

LME(lme.lu) is an ambitious school entrepreneurship project that we're transforming into a real business. What started as an academic initiative is now evolving into a full-fledged startup, combining our learning experience with real-world business development.

LME provides a Secure Multi-Tenant Hosting Platform built on Hetzner infrastructure, designed to offer small organizations and individual clients reliable web hosting without the complexity and cost of large cloud providers.

Visit LME Website

Business Vision

LME bridges the gap between complex enterprise cloud solutions and unreliable shared hosting. Our mission is to provide:

  • Professional Hosting Solutions - Enterprise-grade infrastructure at accessible prices
  • Complete Isolation - Each client website runs in its own secure environment
  • Transparent Operations - Clear documentation and straightforward management
  • Reliable Service - 24/7 monitoring with independent availability checks

Technology Stack

Infrastructure & Virtualization

  • Hetzner Dedicated Server - Reliable German infrastructure provider
  • Proxmox - Enterprise-grade virtualization platform for VM management
  • Docker + Docker Compose - Container orchestration for service isolation

Web Services & Security

  • Nginx - High-performance web server and reverse proxy
  • Cloudflare - DNS management, DDoS protection, and security features
  • SSH & Firewall - Secure access and network protection

Monitoring & Management

  • Raspberry Pi - Independent external monitoring system
  • Hetzner API - Resource monitoring and server management
  • Custom Scripts - Automated availability checks and alerts

Technical Architecture

LME's infrastructure is built on a multi-layered approach that ensures security, isolation, and scalability:

Multi-Tenant Isolation

  • VM-Level Separation - Each client website runs in a dedicated Proxmox virtual machine
  • Container Isolation - Docker Compose separates services within each VM
  • Network Segmentation - Firewall rules prevent cross-tenant access
  • Resource Allocation - Guaranteed CPU, RAM, and storage per tenant

Security Measures

  • Cloudflare Protection - DDoS mitigation and web application firewall
  • Secure Access - SSH key-based authentication with no password access
  • Automated Updates - Regular security patches and system updates
  • Backup Strategy - Regular VM snapshots for disaster recovery

External Monitoring

  • Independent Checks - Raspberry Pi monitors from outside the hosting network
  • API Integration - Hetzner API provides real-time resource metrics
  • Availability Tracking - Continuous uptime monitoring for all client sites
  • Automated Alerts - Immediate notifications for service disruptions

Project Scope & Implementation

Project Phases

  • Phase 1: Planning & Setup (6 hours) - Infrastructure design and server provisioning
  • Phase 2: Virtualization (12 hours) - Proxmox installation and VM configuration
  • Phase 3: Service Deployment (8 hours) - Dockerized web services and client onboarding
  • Phase 4: Monitoring & Documentation (12 hours) - External monitoring setup and maintenance guides

Success Criteria

  • Multiple client websites running reliably and independently
  • Monitoring system accurately reporting availability and resource status
  • Secure access and DNS configuration verified and documented
  • Clear documentation enabling repeatable deployment and maintenance

Problem Solved

Many small projects and organizations face a hosting dilemma: enterprise cloud platforms are complex and expensive, while budget shared hosting lacks reliability and isolation. Setting up proper hosting infrastructure requires expertise in security, virtualization, and monitoring—knowledge that's often beyond the reach of beginners.

LME solves this by providing enterprise-grade infrastructure with straightforward management, making professional hosting accessible to small clients who need reliability without complexity.

Project Impact

LME demonstrates the practical application of infrastructure management, cloud hosting concepts, and entrepreneurial thinking. The project showcases skills in virtualization, containerization, networking, security, and business development—all while creating a real service that addresses genuine market needs.

This entrepreneurship journey combines technical expertise with business acumen, proving that school projects can evolve into viable commercial ventures with the right support, vision, and execution.
